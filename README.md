1. For running smart contract(requires truffle and ganache)
 In the terminal:
 cd smart_contract 
 truffle compile
 truffle migrate


2. For server(requires MongoDB Compass and local server running in background) 
  In the terminal:
   npm install
   nodemon main


3. For client
   In the terminal:
   npm install
   npm run start

 Wait for few minute for the website to load.
 Viola!
